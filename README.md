# MarTech Web Application

This folder contains the complete source code for the MarTech Web Application.

## Getting Started

Since the project was initialized manually, you need to install the dependencies before running it.

### 1. Install Dependencies
Open a terminal in this `webapp` folder and run:
```bash
npm install
```

### 2. Run Development Server
Start the local server:
```bash
npm run dev
```

### 3. Open in Browser
The terminal will show a local URL (usually `http://localhost:5173`). Open this link in your browser.

## Features Implemented
- **Authentication**: Login and Register pages with modern dark design.
- **Dashboard**: Sidebar navigation, Top bar, and Overview page.
- **Brand Analysis**: SWOT analysis view.
- **Roadmap**: Visual timeline of brand progress.
- **Design System**: Fully integrated with the Landing Page's aesthetics (Dark mode + Amber accent).
